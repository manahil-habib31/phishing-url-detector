# Phishing URL Detector

ML-based system to classify URLs as phishing or legitimate using feature engineering and classical ML models (Logistic Regression, Random Forest, XGBoost).

## Project Status: 🚧 In Progress (Phase 0 - Setup)

## Tech Stack
- Python, scikit-learn, XGBoost, pandas

## Project Structure

phishing-url-detector/
├── data/           # Raw and processed datasets
├── notebooks/      # Jupyter notebooks for exploration
├── src/            # Production-ready scripts
├── models/         # Saved trained models
└── app/            # Flask demo app (optional)
